# Cut Tracker Pro — No Supabase Version

A local-first, no-backend rebuild of Cut Tracker Pro based on the screenshots.

## Uses
- HTML
- CSS
- JavaScript
- localStorage
- Vercel static hosting

## Does not use
- Supabase
- login
- database
- server

## Features
- Daily check-in
- 4 habit toggles
- Current streak / best streak / today score
- Progress ring
- Streak calendar
- 12-week goal countdown
- Recent check-ins
- Weight trend chart
- Export data
- Clear all data

## Deploy to Vercel
1. Upload this folder to GitHub
2. Import the repo into Vercel
3. Deploy
4. No environment variables are required

## Notes
- Data saves only on the current browser/device
- Export your data if you want a backup
